Ce projet dépend des technologies et ressources suivantes \+:


\begin{DoxyItemize}
\item Langages \+:
\begin{DoxyItemize}
\item {\ttfamily HTML 5}, {\ttfamily CSS 3}, et {\ttfamily Javascript}
\item {\ttfamily Python}
\item {\ttfamily C++} compilé avec {\ttfamily g++}
\end{DoxyItemize}
\item Formats de fichiers \+:
\begin{DoxyItemize}
\item \href{https://fr.wikipedia.org/wiki/JavaScript_Object_Notation}{\texttt{ {\ttfamily JSON}}} majoritairement
\item \href{https://fr.wikipedia.org/wiki/GeoJSON}{\texttt{ {\ttfamily GEOJSON}}} également, parfois plus adapté
\item \href{https://fr.wikipedia.org/wiki/Comma-separated_values}{\texttt{ {\ttfamily CSV}}} pour explorer les données de manière plus convéniente avec Excel
\end{DoxyItemize}
\item Ressources, plugins, packages \+:
\begin{DoxyItemize}
\item \href{https://www.openstreetmap.org}{\texttt{ {\ttfamily Open\+Street\+Map}}} pour les fonds de carte
\item \href{https://data.iledefrance.fr/}{\texttt{ Base Open\+Data Île de France}} et notamment les bases de données suivantes \+:
\begin{DoxyItemize}
\item \href{https://data.iledefrance.fr/explore/dataset/base-sirene}{\texttt{ Base Sirene des entreprises et de leurs établissements}}
\item \href{https://data.iledefrance.fr/explore/dataset/base-adresse-75}{\texttt{ BAN -\/ Base Adresse Nationale -\/ Paris}}
\item \href{https://data.iledefrance.fr/explore/dataset/base-adresse-92}{\texttt{ BAN -\/ Base Adresse Nationale -\/ Hauts-\/de-\/\+Seine}}
\item \href{https://data.iledefrance.fr/explore/dataset/base-adresse-94}{\texttt{ BAN -\/ Base Adresse Nationale -\/ Val-\/de-\/\+Marne}}
\item \href{https://data.iledefrance.fr/explore/dataset/base-adresse-94}{\texttt{ BAN -\/ Base Adresse Nationale -\/ Val-\/de-\/\+Marne}}
\item \href{https://data.iledefrance.fr/explore/dataset/base-adresse-93}{\texttt{ BAN -\/ Base Adresse Nationale -\/ Seine-\/\+Saint-\/\+Denis}}
\end{DoxyItemize}
\item Les plugins Javascript \+:
\begin{DoxyItemize}
\item \href{https://leafletjs.com/}{\texttt{ {\ttfamily Leaflet}}} pour insérer des cartes OSM dans les pages web
\item le projet \href{https://github.com/Leaflet/Leaflet.markercluster}{\texttt{ {\ttfamily Leaflet/\+Leaflet.\+markercluster}}} pour regrouper les points et accélèrer l\textquotesingle{}affichage
\item le projet \href{https://github.com/pointhi/leaflet-color-markers}{\texttt{ {\ttfamily pointhi/leaflet-\/color-\/markers}}} pour des marqueurs de couleurs variées
\end{DoxyItemize}
\item Les packages Python \+:
\begin{DoxyItemize}
\item \href{https://geopandas.org/}{\texttt{ {\ttfamily Geo\+Pandas}}} pour analyser et traiter les données géographiques
\item \href{https://python-visualization.github.io/folium/}{\texttt{ {\ttfamily Folium}}} pour générer des cartes et fichiers {\ttfamily HTML}
\item \href{https://docs.python.org/fr/3/library/json.html}{\texttt{ {\ttfamily json}}} pour traiter des fichiers {\ttfamily JSON}
\item \href{https://pypi.org/project/ijson/}{\texttt{ {\ttfamily ijson}}} pour traiter de manière itérative de lourds fichiers {\ttfamily JSON}
\item \href{https://docs.python.org/fr/3/library/time.html}{\texttt{ {\ttfamily time}}} pour mesurer le temps de traitement
\item \href{https://matplotlib.org}{\texttt{ {\ttfamily matplotlib}}} pour analyser les données issues des bases ainsi que visualiser le résultat du clustering
\item \href{https://jupyter.org/}{\texttt{ {\ttfamily Jupyter}}} pour développer de manière plus rapide (supporte même {\ttfamily Folium})
\item \href{https://www.riverbankcomputing.com/software/pyqt/}{\texttt{ {\ttfamily Py\+Qt5}}} pour réaliser l\textquotesingle{}interface homme-\/machine
\item \href{https://wiki.qt.io/QtWebEngine}{\texttt{ {\ttfamily Qt\+Web\+Engine}}} pour afficher les fichiers {\ttfamily HTML} générés par {\ttfamily Folium} dans l\textquotesingle{}interface {\ttfamily Py\+Qt}
\item \href{https://cython.org/}{\texttt{ {\ttfamily Cython}}} pour compiler certains de nos modules et accélérer notre code
\end{DoxyItemize}
\item Les librairies {\ttfamily C++} \+:
\begin{DoxyItemize}
\item {\ttfamily iostream} pour les entrées/sorties
\item {\ttfamily fstream} pour lire/écrire les fichiers
\item {\ttfamily string} pour manipuler les chaînes de caractères
\item {\ttfamily chrono} pour mesurer le temps d\textquotesingle{}exécution des différentes parties du programme 
\end{DoxyItemize}
\end{DoxyItemize}
